## 🧬 About Me

I am a **bioinformatician and data scientist** specializing in multi-omics data analysis, genomics, and computational biology.

My work focuses on:
- Analyzing high-throughput sequencing data (RNA-seq, variant analysis, metagenomics)
- Developing reproducible bioinformatics pipelines
- Applying machine learning to biological datasets
- Building tools for data visualization and interpretation

I am particularly interested in **aging biology, disease mechanisms, and genotype–phenotype relationships**, where data-driven approaches can unlock new biological insights.
